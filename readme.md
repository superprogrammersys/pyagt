PyAGT Library
Features
• 
Audio game development features, including:
• 
Real sound: 1D, 2D, 3D
• 
TTS events
• 
Keyboard events
• 
Online networking
Note
• 
Some features, including networking, will be added in the future, so you may not be able to use them currently.
Usage
1. 
Copy the pyagt folder to your project directory.
2. 
Install the required packages using the following command:
Windows/Linux:
pip install -r requirements.txt
Linux (Python3):
pip3 install -r requirements.txt
3. 
It is recommended to use a virtual environment to isolate your work:
python -m venv .env
4. 
To activate the virtual environment:
Windows:
.env\Scripts\activate
Mac/Linux:
source .env/bin/activate
